# QLeverToWebAssembly
This repo contains a project which starts a simple, locally hosted website. On this site you can query prebuilt indexes using the SPARQL language.
This is done with the port from [QLever](https://github.com/ad-freiburg/qlever) to WebAssembly using Emscripten.

Simply checkout this repo and do one of the following:

Run the dockerfile

OR

install npm, nvm and nodejs > v20. Then run `npm run dev` in the root folder of the project.
